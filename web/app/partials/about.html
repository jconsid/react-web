Om applikationen

Applikationen tillhandahåller enkel funktionalitet för att skapa, uppdatera och visa anmälningar om stöld. Funktionaliteten är reducerad till att endast visa en del viktiga kocept.

Namedropping av tekniska beståndsdelar
  • Java & JavaScript
  • vert.x
  • AngularJS
  • SockJS
  • Twitter Bootstrap (tema flatly)
  • mongodb
  • git & GitHub
  • maven (för java byggprocess)
  • grunt (för javascript byggprocess)
  • karma och jasmine (för enhetstester)
  • JSHint (för javascript kodstandard)
Instruktioner för att köra

Checka ut från GitHub och kör igång med följande kommando

vert.x run app.js

Kör javascript-test och linting enligt fölande

npm install
grunt watchKör js-tester kontinuerligt
grunt jshintGör lint
grunt buildPaketera (minifiera och konkatenera js-filer)
