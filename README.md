react-web
=========

Installera för att köra:
- MongoDB
- vert.x
- Maven

Ett vert.x-projekt som initierar moduler, bl a:
mod-web-server
mod-auth-mgr
mod-mongo-persistor
com.consid.react~reactive-module~0.1

Projektet innehåller även (alla?) statiska web-resurser.

Kör igång med:
"vertx run app.js""

eller

"vertx run app.js -conf conf.json"
